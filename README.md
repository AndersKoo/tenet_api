## tenet_api
Dette er et python basert prosjekt med et egenutviklet API.

## Demonstrasjon
 ![Tenet](demo.gif)

## Instrukser
Windows: I terminal <br> 
`python3 -m venv .venv`<br>
`cd api `<br>
`cd .venv`<br>
`Scripts\activate`<br>
`pip3 install flask-sqlalchemy`<br>
`cd..`<br>
`set FLASK_APP=app.py`<br>
`set FLASK_ENV=development`<br>
`flask run`<br>
